# Chart Dashboard

> Dashboard with auth and line chart graphic

## Build Setup

``` bash
# install dependencies
$ npm install

Generate JWT_SECRET in env file

# serve with hot reload at localhost:3000
$ nuxt

# build for production
$ nuxt build
$ nuxt start

# generate static project
$ nuxt generate
```
